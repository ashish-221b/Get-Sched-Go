Detailed Description

A model named Event.

The most elementary model of the project.It contains all the fields that specifies all the details of an event. CreatorType corresponds to what type of event an instructor is creating i.e, assignments, classes, exams etc. Creatorid correspond to the id of the respective type of event(model). Completed is a booleanfield which stores whether the event is completed or not.
