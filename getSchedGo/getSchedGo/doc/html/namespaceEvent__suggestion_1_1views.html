A view which converts the suggestion into an event.

Parameters
Request,SuggestionId

First it gets the object with the suggestion id.Next it calls the constructor of creating the an Event model instance and the fields are filled up in accordance with the fields of the corresponding suggestion model instance. The Event is added to the database and then it redirects you to EditEvent view of Timetable for editing the newly created event.
