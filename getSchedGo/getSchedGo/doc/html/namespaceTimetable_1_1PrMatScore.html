def Timetable.PrMatScore.scoreCalc (   eventType,
  slotNumber,
  user 
)

A method for calculating score according to slotno., eventType, and user preference.

Parameters
eventType,slotNumber,user

There are three matrices for three different kinds of users, i.e. user who prefers to study at day, people who prefer night and people who prefer Latenight. Each matrix contains five rows, each row corresponds to a specific eventtype and contains 48 scores for 48 slots . The function returns the corresponding score.
